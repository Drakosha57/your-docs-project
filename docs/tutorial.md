# Создайте свой первый кулинарный сайт за 30 минут

Добро пожаловать в Foodieloog! Это пошаговое руководство поможет вам создать и запустить собственный кулинарный сайт, даже если вы никогда раньше не занимались веб-разработкой.

## Цель этого руководства

К концу этого руководства вы сможете:
- Скачать и запустить готовый кулинарный сайт
- Изменить главную страницу под себя
- Добавить свой собственный рецепт
- Опубликовать сайт в интернете

## Что вам понадобится

- Компьютер с Windows, MacOS или Linux
- Доступ в интернет
- Текстовый редактор (Блокнот на Windows или TextEdit на Mac тоже подойдут)
- Желание научиться!

## Шаг за шагом

### Шаг 1: Скачивание проекта

1. **Откройте браузер**
2. **Перейдите** по адресу: `https://github.com/ваш-username/your-docs-project`
3. **Найдите зеленую кнопку** "Code" и нажмите на нее
4. **Выберите** "Download ZIP"
5. **Сохраните** файл на рабочий стол или в другую удобную папку
6. **Разархивируйте** скачанный файл:
   - На Windows: щелкните правой кнопкой → "Извлечь все"
   - На Mac: просто дважды щелкните по файлу

**Проверка**: У вас должна появиться папка `foodieloog-master` с файлами внутри.

### Шаг 2: Знакомство со структурой

1. **Откройте папку** `src`
2. **Посмотрите на файлы** внутри:
   - `Главная.html` - это главная страница сайта
   - `Рецепты.html` - страница с рецептами
   - `Ужин.html` - статьи о правильном питании
   - `Десерты.html` - рецепты десертов
   - `Меню.html` - примеры меню
   - `О нас.html` - информация о проекте
   - `контакты.html` - контакты и обратная связь

### Шаг 3: Запуск сайта локально

1. **Найдите файл** `Главная.html` в папке проекта
2. **Щелкните правой кнопкой** мыши по файлу
3. **Выберите** "Открыть с помощью" → ваш браузер (Chrome, Firefox и т.д.)
4. **Дождитесь** загрузки страницы

**Ура!** Вы только что запустили свой первый веб-сайт!

### Шаг 4: Навигация по сайту

1. **Посмотрите** на верхнюю часть страницы - там находится меню
2. **Нажмите** на пункт "Рецепты" - вы перейдете на другую страницу
3. **Нажмите** на пункт "Дом" - вернетесь на главную
4. **Попробуйте** перейти на другие страницы:
   - Ужин
   - Десерты
   - Меню
   - О нас
   - Контакты

### Шаг 5: Ваша первая правка - изменение заголовка

2. **Щелкните правой кнопкой** по файлу `Главная.html`
