## Быстрый старт за 5 минут

### Предварительные требования

- Веб-браузер (Chrome, Firefox, Safari, Edge)
- Любой текстовый редактор (VS Code, Sublime Text)
- Git (опционально)

### Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ваш-username/your-docs-project.git
cd your-docs-project

# 2. Откройте главную страницу в браузере
# На Windows:
start Главная.html

# На macOS:
open Главная.html

# На Linux:
xdg-open Главная.html
